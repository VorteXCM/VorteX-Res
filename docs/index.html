مقدمه

به وب‌سایت VorteX Rex خوش آمدید! این سایت یک مرجع کامل برای تمام علاقه‌مندان به VMP،Fivem،IRfive است. هدف ما ارائه‌ی ریسورس‌ها، اسکریپت‌ها، مپ‌ها و سرورهای آماده در یک مکان است تا بتوانید به راحتی آن‌ها را دانلود و استفاده کنید. 

# نمونه کد نصب ریسورس
1. فایل zip ریسورس را دانلود کنید
2. آن را در پوشه resources سرور خود قرار دهید
3. نام ریسورس را در server.cfg اضافه کنید:

start my_resource

4. سرور را ریستارت کنید

شما می‌توانید از لینک‌های زیر برای دسترسی سریع به بخش‌های مختلف استفاده کنید:

  • LoadingScreen - دانلود و راهنمای نصب صفحه لودینگ
  • سرور کامل نوپیکسل - سرور آماده با کانفیگ کامل
  • مپ‌ها - مجموعه‌ای از شهرها و محیط‌های آماده
  • اسکریپت‌ها - پلاگین‌ها و ماژول‌های مختلف

VorteX Rex تلاش می‌کند تا تمامی ابزارها و آموزش‌ها را در یک مکان جمع‌آوری کند تا شما بدون نیاز به جستجوی زیاد، بتوانید سرورهای شخصی بسازید و تجربه‌ی کامل VMP،Fivem،IRfive را داشته باشید. 

# مثال تنظیمات سرور
sv_hostname "My VMP،Fivem،IRfive Server"
rcon_password "12345"
start mapmanager
start chat
start spawnmanager

از شما دعوت می‌کنیم که تمام بخش‌های سایت را بررسی کنید و با استفاده از منابع ارائه شده، دنیای VMP،Fivem،IRfive را به شکل حرفه‌ای تجربه کنید.
